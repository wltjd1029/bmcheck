⚙ 배정물량 설정 (구글시트 연동)

설정값은 구글시트 설정시트 탭에서 불러옵니다. 값 변경은 시트에서 하세요. 배정물량 = (해당 요일 1세트 배정건수) × (세트 수)

세트 수 (전체 공통) - — 설정시트의 '세트수' 행에서 불러옴
📄 구글시트 열기
구글시트 탭 구성 안내
• 부평현황시트 탭: 완료콜/보조 데이터 (아침점심피크완료 / 오후논피크완료 / 저녁피크완료 / 심야피크완료 / 운행중인원수 / 거절 / 배차취소 / 배달취소(라이더귀책))
• 설정시트 탭: 1행 헤더 = 구분, 수, 목, 금, 토, 일, 월, 화 / 2~5행 = 각 파트 요일별 1세트 배정건수 / 별도 행 '세트수' = 전체 세트 수
